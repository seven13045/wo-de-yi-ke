「我的一课」隐私政策

更新日期：2026年6月1日 · 生效日期：2026年6月1日

「我的一课」（以下简称「本 App」）是一款帮助教师查看与管理个人课表的工具。我们重视您的隐私。本政策说明本 App 如何处理信息。

1. 无需注册

本 App 不要求注册、登录或提供姓名、邮箱等账号信息。开发者不运营用户账号系统，也不将课表数据上传至开发者服务器。

2. 本地存储的数据

您导入或录入的课表、实训安排、学期与校历设置保存在设备本地（Application Support）。显示偏好保存在设备 UserDefaults 中。卸载 App 将删除上述本地数据。

3. 照片与文件

当您导入课表时，本 App 可能访问您主动选择的 PDF 或照片，用于解析课程信息。

  • 本地识别：使用本机 Vision 识别截图时，照片不会上传至开发者或第三方服务器。
  • AI 识图（可选）：若您在设置中配置阿里云 DashScope API Key 并使用该功能，您选择的照片将由您的设备直接发送至阿里云进行处理。该功能完全可选；API Key 由您自行申请并保存在本机钥匙串。请参阅阿里云服务条款与隐私说明。

4. 网络通信

除您主动使用 AI 识图外，本 App 核心功能可在离线状态下使用。本 App 不包含广告，不进行跨 App 追踪。

5. 第三方服务

  • 阿里云 DashScope：仅在您启用 AI 识图时，由您的设备直接调用。
  • Apple App Store：应用分发与更新由 Apple 提供。

6. 儿童隐私

本 App 不面向 13 岁以下儿童，也不会故意收集儿童个人信息。

7. 政策变更

我们可能更新本政策，并在本页面修改「更新日期」。
